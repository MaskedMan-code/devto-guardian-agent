# 🏴‍☠️ The Coral Lookout: Autonomous Intelligence Scout

![Track](https://img.shields.io/badge/Track-Personal_Agent_(Track_2)-blue)
![Bounty](https://img.shields.io/badge/Bounty-Chart_New_Waters-gold)

A swift, self-contained personal AI agent built single-handedly for **Pirates of the Coral-bean**. 

The Coral Lookout acts as a digital bodyguard for solo developers navigating external blog ecosystems. It automatically scouts public developer waters (Dev.to API), sifts "Pure Gold" educational content from "Poisoned Booty" (malicious npm packages, phishing schemes, wallet drains), and maps them onto an elegant, chronological activity timeline.

## 🗺️ Chart New Waters Bounty Eligibility
This vessel carries a hand-crafted custom pipeline connector (`coral/reddit_guardian_connector.yaml`). It maps previously uncharted web API territory (Dev.to) into standard Coral SQL, allowing the agent to dynamically extract and cross-reference live threat signatures across the web without traditional scraping overhead.

## ⚙️ The Ship's Engineering (Tech Stack)
* **The Cartographer's Ledger (Data Layer):** Coral CLI (Custom Dev.to YAML Source Connector)
* **The Deep-Sea Oracle (AI Engine):** Google Gemini 2.5 Flash (for zero-day threat analysis)
* **The Captain's Deck (Frontend):** Python & Streamlit (Custom dark-cyber UI)
* **The Shipyard (Deployment):** Docker & Railway

## 🏴‍☠️ Core Features
* **Autonomous Scouting:** Converts Dev.to API responses into SQL tables via Coral.
* **Siren Trap Detection:** Gemini AI analyzes payload headers and text descriptions to flag high-risk links.
* **Chronological Thought Stream:** Replaces flat dashboards with a proactive, chronological agent activity log.
